## 🌐 Project Ke Baare Mein

Bahut saari bot scripts available hain, par kuch hi sachi mein customizable, hybrid aur **Anti-Bot secured** hain. **OP_INJOY V4** ek fully-fledged bot swarm tool hai jo Android (Termux), Linux Servers (Debian/Ubuntu), aur Windows par makhan ki tarah chalta hai. 

### 🚀 Ultimate V4 Features:
* **10-Step Smart Wizard**: Ek proper software UI jisme tum kisi bhi question par `back` type karke peeche jaa sakte ho bina script restart kiye.
* **Auto IP & Port Parsing**: IP aur Port ko `domain:port` format me dalo, script automatically Java aur Bedrock ke liye usko optimize kar legi.
* **Smart Password Generator**: Agar password khali chhodte ho, toh bot automatically (Name + Word + Number) ka ek super secure unique password bana lega.
* **Default Version Fallback**: Tumhe baar-baar version dalne ki zaroorat nahi. Auto mode par Java `1.21.4` aur Bedrock `1.26.30` auto-select ho jata hai.
* **Advanced Player Tracking**: Naya `Follow Player` mode! Bas player ka naam dalo aur bot uske peeche-peeche ghoomta rahega.

### ⚡ 1. Linux & Termux Setup (Install / Update)
```bash
curl -fsSL raw.githubusercontent.com/opinjoy7055/OPINJOY_/main/install.sh | bash

```

### 🪟 2. Windows Setup (Install / Update)
> **IMPORTANT: ⚠️ OPEN POWERSHELL AS ADMINISTRATOR! ⚠️**
